> **Module 1 · Python Programming Fundamentals and Flow Control**

## What you'll learn

- Initialise a Python environment in **Google Colab** — open a notebook, run a cell, see output
- Use Python's basic data types: `int`, `float`, `str`, `bool`
- Store and rename values using **variables**
- Manipulate **strings** — concatenate, slice, transform with methods, build with f-strings
