# Session 3.1 — Iteration and Loop Mastery

> **Module 1 · Python Programming Fundamentals and Flow Control**

In Session 2.2 your code learned to make a decision on **one** value. Today it learns to make that same decision on **a thousand** values — automatically. Welcome to loops.

---

## What you'll learn

- Use a **`for`** loop to walk through every item in a list, string, dict, or set
- Generate number sequences with **`range()`**
- Use a **`while`** loop when you don't know how many times to repeat
- Skip or stop early with **`continue`** and **`break`**
- Build the three classic loop patterns: **accumulator**, **counter**, **filter**
- Combine loops with the `if`/`else` you learned in 2.2
- Avoid the two scariest beginner errors: infinite loops and modifying a list while iterating
