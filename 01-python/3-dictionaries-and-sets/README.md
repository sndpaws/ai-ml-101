# Session 2.1 — Advanced Data Structures (Dictionaries & Sets)

> **Module 1 · Python Programming Fundamentals and Flow Control**

In Session 1.2 you stored many values **by position** (`students[0]`, `students[1]`). Today you'll store them **by name** — and learn a structure where every item is guaranteed unique.

---

## What you'll learn

- Store data as **key-value pairs** using **dictionaries** `{ "key": value }`
- Look up values instantly by key, and use **`.get()`** to handle missing keys safely
- Add, update, and remove keys; iterate over `.keys()`, `.values()`, `.items()`
- Use **sets** `{ 1, 2, 3 }` to keep only unique items
- Combine groups with set math — **union `|`**, **intersection `&`**, **difference `-`**
- Choose between list, dict, and set for a given problem, and explain why
