# Session 1.2 — Core Python Data Structures (Lists & Tuples)

> **Module 1 · Python Programming Fundamentals and Flow Control**

In Session 1.1 you stored **one** value in a variable. Today you'll store **many**.

---

## What you'll learn

- Store many values in one variable using **lists** `[ ]` and **tuples** `( )`
- Access items by **indexing** and ranges by **slicing**
- **Modify** a list — `append`, `insert`, `remove`, `sort` (and the `.sort()` returns `None` trap)
- Use **tuples** for fixed records and unlock **tuple unpacking**
- Choose between a list and a tuple for a given problem, and explain why
