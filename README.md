# AI / ML 101 — IIT Reict Cohort 2604

A **comprehensive Artificial Intelligence and Machine Learning program** covering Python fundamentals, data handling, statistical thinking, classical ML, and modern Generative AI — culminating in an end-to-end capstone project.

> **Delivered by:** Industry Mentors and Professors

---

## How this repo is organized

Each session is its own folder. **Click the session title to land on a short overview page** — it links to everything below in order.

Inside each session folder:

| Path | Purpose | When to look |
|---|---|---|
| `README.md` | Session overview — start here | **First**, every session |
| `pre-class/` | Reading + setup before class | **Before** the live class |
| `live-class/` | Topics list + code we wrote together | **After** the lecture is pushed |
| `post-class/` | Drill problems, mini-builds, reflection | **After** class, for practice |
| `questions.md` | Question set used for Mentimeter polls — your mentor shares the live link | During / after class |
| `assets/` | Images and visual aids referenced from the notes | Linked inline where used |
